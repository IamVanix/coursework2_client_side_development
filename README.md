Spice Haven - Restaurant Website
This is a modern, responsive website for Spice Haven, developed for the COM109 Coursework 2. It features a clean design with interactive elements using HTML, CSS, and JavaScript.

Key Features
Dynamic Menu: Users can filter food items by "Veg" or "Non-Veg" categories.
Booking Form: A functional contact page where customers can book a table.
Animations: Uses the AOS (Animate On Scroll) library for smooth visual effects.
Modern Design: Built with Bootstrap 5 and custom "Glassmorphism" styles.

🛠️ Built With
HTML5 & CSS3: For structure and layout styling.
JavaScript: Handles menu filtering and form validation.
Bootstrap: Used for responsive navigation and grid layouts.
GitHub: Used for version control and project management.

Project Pages
Home (index.html): Features a hero section and "Today's Special" button.
Menu (menu.html): Interactive food list with price tags and favorite buttons.
About (about.html): Information about the restaurant's flavors.
Contact (contact.html): Form for table reservations.

How to Run
Simply open the index.html file in any web browser to view the site.
